# AI SEO Land

Стартовая структура для демо-лендинга.

## Структура

```text
.
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
└── assets/
    ├── icons/
    └── images/
```

## Запуск

Откройте `index.html` в браузере или запустите локальный статический сервер:

```bash
python3 -m http.server 8000
```
